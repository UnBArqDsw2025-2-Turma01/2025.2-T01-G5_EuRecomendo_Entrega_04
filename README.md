# EuRecomendo - Entrega 04

Repositório da quarta entrega do projeto EuRecomendo, contendo a documentação de arquitetura de software (DAS).

## Arquitetura e Reutilização de Software

Esta entrega foca na definição e documentação da arquitetura do sistema, seguindo o modelo 4+1 do RUP e aplicando as melhores práticas de reutilização de software.

## Tecnologia

A geração do site estático é realizada utilizando o [docsify](https://docsify.js.org/).

### Executando localmente

Para iniciar o site localmente, utilize o comando:

```shell
docsify serve ./docs
